How to Use Whitesnake

Follow the walkthrough to see exactly what happens from first launch to green checkmarks. The first five steps capture the primary flow; the gallery showcases the remaining screens so you know what to expect.

1

Launch Whitesnake

Open Whitesnake from Applications (or Spotlight). The main window opens and immediately starts scanning your environment state.

2

Confirm First-Run Prompts

macOS may prompt to confirm that you trust the download. Approve the dialog so the helper processes can launch.

3

Review Environment Checks

The overview lists every tool Whitesnake audits. Each row highlights the current status and the action that will be taken if something is missing.

4

Patch Everything at Once

Checks run automatically on launch. Press Patch to fix all failing items in one go, or click Fix on individual rows to handle them one at a time.

5

Fix Issues Instantly

If a check fails, click Fix. Whitesnake runs the recommended command for you or deep-links to the relevant installer.
