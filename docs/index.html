How to Use Whitesnake

Follow the walkthrough to get from download to a fully verified environment.

1

Download Whitesnake

Visit the website and click Download Whitesnake. The latest DMG will be saved to your Downloads folder.

2

Open the DMG

Double-click Whitesnake-x.x.x.dmg in your Downloads folder to mount the disk image.

3

Drag to Applications

Drag the Whitesnake icon onto the Applications folder shortcut inside the DMG window to install it.

4

Launch from Applications

Open your Applications folder, find Whitesnake, and double-click to launch it.

5

Bypass the macOS Security Warning

macOS will block the app on first launch because it is not from the App Store. Click Done, then go to System Settings → Privacy & Security and click Open Anyway.
