First-time setup

Since Whitesnake is an open-source personal project, it is not notarized by Apple. macOS may say the app is "damaged" — this is a misleading message for unsigned apps. Here's how to fix it:

  1. Open the DMG and drag Whitesnake.app into your Applications folder
  2. Open Terminal and run:
xattr -cr /Applications/Whitesnake.app
  1. Open Whitesnake from Applications

The -cr flag recursively clears all quarantine attributes. You only need to do this once.

Requirements

  • macOS — 14 (Sonoma) or later
  • Architecture — Apple Silicon (arm64)
  • Prerequisite — Xcode Command Line Tools (xcode-select --install)
